# AutoGen

AutoGen is a framework for building multi-agent systems where agents can converse, collaborate, and use tools to solve tasks.

## Why It Matters

AutoGen is useful when you want to explore agent-to-agent interaction patterns such as:

- planner and executor workflows
- assistant and reviewer loops
- multi-model collaboration
- code and task automation through agent conversations

## Core Concepts

- agents with distinct roles
- message passing between agents
- tool and code execution support
- human-in-the-loop participation when needed

## When I Would Use It

- when I want to prototype conversational multi-agent systems
- when collaboration between specialized agents is the main design pattern
- when I want to compare agent dialogue against graph or flow orchestration

## What I Should Learn Next

1. How to define agent roles clearly
2. How to reduce unnecessary back-and-forth between agents
3. How to add tools safely
4. How to evaluate whether multi-agent collaboration actually improves results

## Project Fit

Good portfolio projects for AutoGen include:

- coding assistant teams
- research and review agents
- incident triage and reporting workflows
