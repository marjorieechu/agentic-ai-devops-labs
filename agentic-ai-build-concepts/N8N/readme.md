# n8n

n8n is a workflow automation platform that supports API integrations, business process automation, and AI-powered workflows.

The current n8n documentation includes an `Advanced AI` section for building chatbots, document workflows, and agent-like automations.

## Why It Matters

n8n is valuable because it gives me a fast way to build visible automations without writing everything from scratch.

It is especially useful for:

- workflow automation
- API orchestration
- trigger-based agents
- low-code demos
- operational integrations across tools

## Where It Fits In My Learning

n8n is not the same kind of framework as LangGraph or CrewAI.

- LangGraph is stronger for code-first orchestration
- CrewAI is stronger for agent-team abstractions
- n8n is stronger for operational workflow automation and integrations

## Strong Use Cases

- issue triage workflows
- email and calendar assistants
- ticket routing and summarization
- report generation pipelines
- support or operations automations
